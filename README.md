# Company Technology Foundation Blueprint

## Overview
This project documents how a company should design a secure and scalable AWS foundation before deploying any servers or applications.

## Why This Project Exists
Many cloud failures happen because companies rush into creating servers without proper security, access control, and availability planning.
This project focuses on doing things the correct industry way.

## What Is Covered
- Cloud vs on-prem understanding
- Internet and server fundamentals
- Linux server basics
- Secure access using SSH
- AWS account security
- IAM users, roles, and least-privilege design
- Region and availability zone planning

## Outcome
After this foundation, a company is ready to safely deploy real workloads on AWS.

## Status
Foundation phase completed.
